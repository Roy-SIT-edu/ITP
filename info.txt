Timetable Scheduler - Student-Friendly Code Rundown
===================================================

This file explains the important files changed in the latest improvement pass.
It focuses on the cleaner workflow: focused Excel import, hard-constraint
validation, requirement editing inside Validation, ranked soft constraints,
schedule comparison, manual timetable moves, explanations, availability
snapshots, and smart issue summaries.

Latest workflow update:
  The workflow is now Import -> Validate -> Soft Constraints -> Review -> Export.
  Validation checks hard blockers first. Soft Constraints lets users rank soft
  preferences before generation. Generate Timetable now lives on the Soft
  Constraints tab, not the Validation tab.
